# Crystal Kairos Component Library

Welcome to the Crystal Kairos Storybook. This documentation showcases the reusable UI components used across the application.

## Setup

Components are wrapped with:

- **MUI Theme** – Crystal Kairos design system (primary, accent, typography)
- **i18n** – React-i18next for translations (common, home, auth namespaces)
- **Tailwind CSS** – Utility classes and design tokens from `globals.css`

## Running Storybook

```bash
pnpm run storybook
```

Storybook runs at [http://localhost:6006](http://localhost:6006).

## Building for deployment

```bash
pnpm run build-storybook
```

The static build outputs to the `storybook-static` folder.

## Adding new stories

Create a `*.stories.tsx` file next to your component or in the `stories` folder:

```tsx
import type { Meta, StoryObj } from '@storybook/nextjs-vite'
import { MyComponent } from './MyComponent'

const meta = {
  title: 'Category/MyComponent',
  component: MyComponent,
  tags: ['autodocs'],
} satisfies Meta<typeof MyComponent>

export default meta
type Story = StoryObj<typeof meta>

export const Default: Story = {
  args: { /* props */ },
}
```

## Note on Logo component

The `Logo` component expects `/logo.png` in the `public` folder. Add your logo asset there for the Logo stories to render correctly.
